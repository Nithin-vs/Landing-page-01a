what is data science?

Data science is the domain of study that deals with vast volumes of data using modern tools and techniques to find unseen patterns, derive meaningful information, and make business decisions. Data science uses complex machine learning algorithms to build predictive models.The data used for analysis can come from many different sources and presented in various formats. Data science is the science of analyzing raw data using statistics and machine learning techniques with the purpose of drawing conclusions about that information.
